## 📖 Qué es este proyecto

Este repositorio contiene un **manual estructurado, operativo y aplicado de OSINT** orientado a investigación criminal real.

### ❌ No es
- Un listado de herramientas  
- Un curso introductorio  
- Contenido superficial  

### ✅ Es
- Una **obra de referencia profesional**
- Metodología aplicable a investigaciones reales
- Integración de:
  - Técnica
  - Análisis
  - Marco jurídico
  - Documentación policial

El enfoque es tratar el OSINT como una **disciplina investigadora completa**, no como un conjunto de utilidades.

El contenido se organiza en tres bloques:

- Conceptos
- Marco legal
- Metodología

### 🔹 Investigación por objeto
Procedimientos según entidad investigada:
- Email
- Teléfono
- Username / alias
- Redes sociales
- Infraestructura digital
- Imágenes y geolocalización

### 🔹 Análisis avanzado
- Criptoactivos
- Análisis relacional
- Documentación policial
- Automatización e IA
